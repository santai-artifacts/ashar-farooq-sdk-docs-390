Getting Started

Introduction

MySDK is a lightweight client library that makes it easy to integrate our API into your application. Works in Node.js, Bun, and the browser.

Install

Add the SDK to your project with your package manager of choice.

Terminal 
npm install mysdk

Quick Example

Initialize the client and make your first call in a few lines.

index.ts 
import { MySDK } from 'mysdk';

// Initialize with your API key
const client = new MySDK({
  apiKey: 'sk-...',
});

// Make a request
const result = await client.hello({ name: 'world' });
console.log(result.message); // "Hello, world!"
